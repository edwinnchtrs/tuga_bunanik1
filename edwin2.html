Wireframe Lofi - DxD Squad MLBB Platform

Struktur dasar dan layout untuk platform esports Mobile Legends DxD Squad

1. Halaman Login

Catatan: Halaman pertama yang dilihat user sebelum mengakses dashboard squad.
Logo DxD Squad
Field: ID Game/Username
Field: Password
Checkbox: Ingat Saya
Link: Lupa Password
Button: Login ke Squad
Link: Belum anggota squad? Daftar untuk bergabung

2. Struktur Dashboard Utama

Catatan: Layout utama setelah login berhasil, menggunakan pendekatan 3-kolom.
Menu Utama
  • Dashboard Utama
  • Anggota Squad
  • Match History
  • Jadwal Latihan
  • Hero Pool
    • Komunikasi
  • Squad Chat
  • Strategi & Draft
  • Voice Channel
    • Kompetisi
  • Tournament
  • Scrim Schedule
  • Statistik
    • Nama Squad: DxD Squad
    Team ID & Region
    Statistik Cepat (Win Rate, Jumlah Match, dll)
    Button: Start Scrim
    Konten Dinamis

    Area ini akan berubah berdasarkan halaman yang dipilih dari menu navigasi

    Contoh: Halaman Anggota Squad

    Grid berisi kartu anggota dengan info: nama, role, win rate, rank

    Tournament Mendatang

    Daftar tournament dengan info hadiah dan tanggal pendaftaran

    Catatan Strategi

    Tips draft, rotasi, dan fokus gameplay

    Notifikasi

    Pemberitahuan scrim, tournament, dan aktivitas squad

    3. User Flow

    Catatan: Alur pengguna dari login hingga menggunakan fitur utama.
    1
    User membuka platform → Tampil halaman login
    ↓
    2
    Login dengan ID Game & password → Masuk ke dashboard utama
    ↓
    3
    Dashboard menampilkan overview squad, statistik, dan notifikasi terbaru
    ↓
    4
    User navigasi ke halaman yang diinginkan melalui menu sidebar
    ↓
    5
    Berinteraksi dengan konten: lihat anggota, jadwal, statistik, atau chat

    4. Daftar Halaman & Fitur

    Dashboard Utama
    • Overview squad (anggota aktif, win rate)
    • Statistik cepat (match, rank)
    • Aktivitas terkini (match terbaru)
    • Notifikasi penting
    Anggota Squad
    • Daftar anggota (dengan role & rank)
    • Detail per anggota (win rate, hero pool)
    • Invite member baru
    • Edit role & informasi
    Match History
    • Riwayat match (win/loss)
    • Detail setiap match
    • Statistik hero yang digunakan
    • Analisis performa
    Jadwal Latihan
    • Kalendar latihan
    • Jadwal scrim
    • Daftar peserta
    • Reminder otomatis
    Hero Pool & Draft
    • Daftar hero yang dikuasai
    • Strategi draft
    • Counter pick guide
    • Role assignment
    Squad Chat & Voice
    • Chat room squad
    • Voice channel
    • Broadcast pengumuman
    • Pengingat jadwal

    5. Komponen UI Utama

    Kartu Anggota (Member Card)
    Kartu Match (Match Card)
    Kartu Jadwal (Schedule Card)
    Kartu Hero (Hero Card)
    Statistik Bar (Stats Bar)
    Modal Dialog
    Notifikasi Toast
    Search Bar
